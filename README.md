Animated BMI Calculator

A fun and interactive BMI calculator with animations, emojis, and health-themed floating icons.

Features
- Animated BMI bar with color changes
- Emoji reaction based on BMI category
- Floating health icons in the background
- Responsive and visually engaging

How to Use
1. Enter your height (cm) and weight (kg)
2. Click Calculate BMI
3. Watch the animations and emoji reactions!

Tech Stack
- HTML, CSS, JavaScript
